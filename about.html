Welcome to my To Do App!

To add a task to your list, go to the home page and fill out the form. Task details are not required but can be added if needed.


Use the Quick Add tab if you don't need to fill out information for the task details, priority, or category. The priority will automatically be set to high and the category will be set to "Quick Add".


To view tasks in each different category, click on the "School", "Chores", or "Goals" tab.
